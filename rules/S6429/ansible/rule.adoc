== Sensitive Code Example
[source,yaml]
----
- name: Create Pod
  kubernetes.core.k8s:
    definition:
      apiVersion: v1
      kind: Pod
      metadata:
        name: test
      spec:
        containers:
        - image: k8s.gcr.io/test-webserver
          name: test-container
          volumeMounts:
          - mountPath: /var/run/docker.sock
            name: test-volume
        volumes:
        - name: test-volume
          hostPath:
            path: /var/run/docker.sock # Sensitive
            type: Socket
----

== Compliant Solution
[source,yaml]
----
- name: Create Pod
  kubernetes.core.k8s:
    definition:
      apiVersion: v1
      kind: Pod
      metadata:
        name: test
      spec:
        containers:
        - image: k8s.gcr.io/test-webserver
          name: test-container
----
