==== Noncompliant code example

[source,java,diff-id=1,diff-type=noncompliant]
----
import android.webkit.WebView;

WebView.setWebContentsDebuggingEnabled(true); // Noncompliant
WebView.getFactory().getStatics().setWebContentsDebuggingEnabled(true); // Noncompliant
----

==== Compliant solution

[source,java,diff-id=1,diff-type=compliant]
----
import android.webkit.WebView;

WebView.setWebContentsDebuggingEnabled(false);
WebView.getFactory().getStatics().setWebContentsDebuggingEnabled(false);
----
