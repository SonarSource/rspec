=== Exceptions

The rule ignores:

* block statements
* anonymous functions containing a single statement

[source,csharp]
----
Func<object, bool> item1 = o => { return true; }; // Compliant by exception
Func<object, bool> item1 = o => { var r = false; return r; }; // Noncompliant
----
