The following objects are considered as shared resources:

* a reference to https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/this[this]
* a https://learn.microsoft.com/en-us/dotnet/api/system.type[Type] object
* a https://learn.microsoft.com/en-us/dotnet/csharp/programming-guide/strings/[string] literal or instance

== How to fix it

=== Code examples

==== Noncompliant code example

[source,csharp,diff-id=1,diff-type=noncompliant]
----
void MyLockingMethod()
{
    lock (this) // Noncompliant
    {
        // ...
    }
}
----


==== Compliant solution

[source,csharp,diff-id=1,diff-type=compliant]
----
private readonly object lockObj = new object();
void MyLockingMethod()
{
    lock (lockObj)
    {
        // ...
    }
}
----
