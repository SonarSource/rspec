== Recommended Secure Coding Practices

It's recommended to avoid exposing instances on the Internet by assigning to them a public IP address, unless the instance in running a service designed to be publicly accessible, such as customer portals or e-commerce websites. To communicate with instances in another VPC, consider using https://docs.aws.amazon.com/vpc/latest/peering/what-is-vpc-peering.html[VPC peering].
