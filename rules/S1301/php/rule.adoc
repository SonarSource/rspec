== Noncompliant Code Example

----
switch ($variable) {
  case 0:
    do_something();
    break;
  default:
    do_something_else();
    break;
}
----

== Compliant Solution

----
if ($variable == 0) {
  do_something();
} else {
  do_something_else();
}
----
