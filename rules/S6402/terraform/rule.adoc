== Sensitive Code Example

----
resource "google_dns_managed_zone" "foo" { # Sensitive: dnssec_config is missing
  name     = "foobar"
  dns_name = "foo.bar."
}
----

== Compliant Solution

----

resource "google_dns_managed_zone" "foo" {
  name     = "foobar"
  dns_name = "foo.bar."

  dnssec_config {
    default_key_specs {
      algorithm = "rsasha256"
    }
  }
}
----
