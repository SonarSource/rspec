=== Noncompliant code example

[source,csharp]
----
using (var d = new Disposable()) // Noncompliant
{
    d.Dispose();
}
----

[source,csharp]
----
using var d = new Disposable();
d.Dispose(); // Noncompliant {{Refactor this code to make sure 'd' is disposed only once.}}
----


=== Compliant solution

[source,csharp]
----
using var d = new Disposable();
----
