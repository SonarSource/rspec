=== Noncompliant code example

[source,cpp]
----
uint8_t  * p1;
uint32_t * p2;
p2 = (uint32_t *)p1; // Noncompliant: Alignments may differ

struct foo {
  int i;
}

int main(int argc, char **argv) {
  struct foo* bar = (struct foo *)main; // Noncompliant
  bar->i = 2;
  return bar->i;
}
----
