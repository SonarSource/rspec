=== Noncompliant code example

[source,cpp]
----
void foo()
{
  int x;
  if (x == 42)
  /* Noncompliant - the following nested block is empty */
  {
  }
  else
  {
    printf("x != 42");
  }
}

void bar()
/* Compliant - functions are not nested blocks */
{
}
----

=== Compliant solution

[source,cpp]
----
void foo()
{
  int x;
  if (x != 42)
  /* Compliant */ 
  {                
    printf("x != 42");
  }
}

/* ... */
----
