== Noncompliant Code Example

----
for (var i:int = 0; i < 42; i++){}  // Noncompliant

try {                               // Noncompliant
} catch (error)
{
    ...
}
----

== Compliant Solution

----
for (var i:int = 0; i < 42; i++);

for (var i:int = 0; i < 42; i++) {
  trace(i);
}
----
