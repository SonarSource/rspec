== Noncompliant Code Example

----
long long1 = 1l; // Noncompliant
float float1 = 1.0f; // Noncompliant
double double1 = 1.0d; // Noncompliant
----

== Compliant Solution

----
long long1 = 1L;
float float1 = 1.0F;
double double1 = 1.0D;
----
