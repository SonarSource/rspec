== How to fix it

=== Code examples

==== Noncompliant code example

[source,csharp]
----
class Person
{
  private int age;

  [Pure] // Noncompliant. The method makes a state change
  void ConfigureAge(int age) =>
    this.age = age;

  [Pure]
  void WriteAge() => // Noncompliant
    Console.WriteLine(this.age);
}
----


==== Compliant solution

[source,csharp]
----
class Person
{
  private int age;

  void ConfigureAge(int age) =>
    this.age = age;
  

  [Pure]
  int Age() =>
    this.age;

  // or remove Pure attribute from the method

  void WriteAge() =>
    Console.WriteLine(this.age);
}
----
