=== Noncompliant code example

[source,cpp]
----
int func(int a, int b) {
  int result = 0;
  a + b; // Noncompliant, no side effect.
  return result;
}
----

=== Compliant solution

[source,cpp]
----
int func(int a, int b) {
  int result = a + b; // Compliant
  return result;
}
----
