== How to fix it

=== Code examples

==== Noncompliant code example

[source,vbnet,diff-id=1,diff-type=noncompliant]
----
Private _foo As New List(Of String) From {"a", "b", "c"}
Private _bar As String() = New String() {"a", "b", "c"}

Public ReadOnly Property Foo() As IEnumerable(Of String) ' Noncompliant: collection _foo is copied
    Get
        Return _foo.ToList()
    End Get
End Property

Public ReadOnly Property Bar() As IEnumerable(Of String) ' Noncompliant: array _bar is copied
    Get
        Return DirectCast(_bar.Clone(), String())
    End Get
End Property
----

==== Compliant solution

[source,vbnet,diff-id=1,diff-type=compliant]
----
Private _foo As New List(Of String) From {"a", "b", "c"}
Private _bar As String() = New String() {"a", "b", "c"}

Public Function GetFoo() As IEnumerable(Of String)
    Return _foo.ToList()
End Function

Public Function GetBar() As IEnumerable(Of String)
    Return DirectCast(_bar.Clone(), String())
End Function
----
