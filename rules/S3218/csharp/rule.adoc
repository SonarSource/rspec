[source,csharp,diff-id=1,diff-type=noncompliant]
----
class Outer
{
  public static int A;

  public class Inner
  {
    public int A; // Noncompliant

    public int MyProp
    {
      get => A; // Returns inner A. Was that intended?
    }
  }
}
----

Here's an example of compliant code after renaming the inner class member:

[source,csharp,diff-id=1,diff-type=compliant]
----
class Outer
{
  public static int A;

  public class Inner
  {
    public int B; // Compliant

    public int MyProp
    {
      get => A; // Returns outer A
    }
  }
}
----
