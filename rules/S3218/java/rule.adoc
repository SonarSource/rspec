[source,text]
----
class Outer
{
  public static int A;

  public class Inner
  {
    public int A; // Noncompliant
    public int MyProp
    {
      get { return A; }  // Returns inner A. Was that intended?
    }
  }
}
----

Here's an example of compliant code after renaming the inner class member:

[source,text]
----
class Outer
{
  public static int A;

  public class Inner
  {
    public int B;
    public int MyProp
    {
      get { return A; }  // Still compiles and runs but functionality has changed
    }
  }
}
----
