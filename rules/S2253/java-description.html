Description

Why is this an issue?

This rule allows banning certain methods.

Noncompliant code example

Given parameters:

  • className:java.lang.String

  • methodName: replace

  • argumentTypes: java.lang.CharSequence, java.lang.CharSequence

String name;
name.replace("A","a");  // Noncompliant

(visible only on this page)

Message

Remove this forbidden call

Parameters

className

Name of the class whose method is forbidden

methodName

Name of the forbidden method

argumentTypes

Comma-delimited list of argument types, E.G. java.lang.String, int[], int

allOverloads

Boolean. Set to true to flag all overloads regardless of parameter type (default: false)
