== Sensitive Code Example

----
String ip = "192.168.12.42"; // Sensitive
Socket socket = new Socket(ip, 6667);
----

== Compliant Solution

----
String ip = System.getenv("IP_ADDRESS"); // Compliant
Socket socket = new Socket(ip, 6667);
----
