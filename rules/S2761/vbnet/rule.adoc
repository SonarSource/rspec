== Why is this an issue?

The repetition of the  ``++Not++`` operator is usually a typo. The second operator invalidates the first one:

[source,vbnet]
----
Dim b As Boolean = False
Dim c As Boolean = Not Not b 'Noncompliant: equivalent to "b"
----
