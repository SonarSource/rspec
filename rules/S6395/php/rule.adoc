== Noncompliant Code Example

[source,php]
----
"(?:number)\d{2}"
----

== Compliant Solution

[source,php]
----
"number\d{2}"
"(?:number)?\d{2}"
----


== Exceptions

This rule does not report an issue if the non-capturing group is an alternation.

[source,php]
----
"(?:number|string)"
----
