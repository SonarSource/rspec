=== Code examples

==== Noncompliant code example

[source,php,diff-id=1,diff-type=noncompliant]
----
function numberOfMinutes($hours) {
  $seconds = 0;   // Noncompliant - $seconds is unused
  return hours * 60;
}
----

==== Compliant solution

[source,php,diff-id=1,diff-type=compliant]
----
function numberOfMinutes($hours) {
  return hours * 60;
}
----

==== Noncompliant code example

[source,php,diff-id=2,diff-type=noncompliant]
----
private function process_request(): {
    $validation_result = $this->validate_input();
    if ($validation_result['is_successful']) {
        $item_data = $validation_result['data']; // Noncompliant - $item_data is unused
    }
}
----

==== Compliant solution

[source,php,diff-id=2,diff-type=compliant]
----
private function process_request(): {
    $this->validate_input();
}
----
