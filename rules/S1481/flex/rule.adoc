== Noncompliant Code Example

[source,flex]
----
public function numberOfMinutes(hours:int):int
{
  var seconds:int = 0;  // seconds is never used 
  return hours * 60;
}
----

== Compliant Solution

----
public function numberOfMinutes(hours:int):int
{
  return hours * 60;
}
----
