Description

Why is this an issue?

The for of statement can only iterate over objects that have an iterator property.

Noncompliant code example

let obj = {};
for(let element of obj) {   // Noncompliant
  ...
}

Compliant solution

let array = [];
for(let element of array) {
  ...
}
