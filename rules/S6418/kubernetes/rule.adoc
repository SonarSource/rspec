== Sensitive Code Example

[source,yaml]
----
apiVersion: v1
kind: Pod
metadata:
  name: app
spec:
  containers:
    - name: waiter
      image: "alpine:latest"
      command: ["/bin/sleep","9000"]
      env:
        - name: api-token
          value: a3gd54hw6a53gds3ha3we5r43h51trg1asf2sdf54g65sd4fg54dfg  # Sensitive
----

== Compliant Solution

[source,yaml]
----
apiVersion: v1
kind: Pod
metadata:
  name: app
spec:
  containers:
    - name: waiter
      image: "alpine:latest"
      command: ["/bin/sleep","9000"]
      env:
        - name: github-token
          valueFrom:
            secretKeyRef:
              name: my-secret
              key: api-key
----
