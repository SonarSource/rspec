== Sensitive Code Example

[source,yaml]
----
apiVersion: v1
kind: Pod
metadata:
  name: nginx-app
spec:
  containers:
    - name: nginx
      image: "nginx:1.21.6"
      ports:
        - containerPort: 80
      env:
        - name: API_TOKEN
          value: "f7a9s8d7f6as98df7a6s9d8f7a6sd9f87as6df"  # Sensitive
----

== Compliant Solution

[source,yaml]
----
apiVersion: v1
kind: Pod
metadata:
  name: nginx-app
spec:
  containers:
    - name: nginx
      image: "nginx:1.21.6"
      ports:
        - containerPort: 80
      env:
        - name: API_TOKEN
          valueFrom:
            secretKeyRef:
              name: my-secret
              key: api-key
----
