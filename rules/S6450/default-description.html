Description

With the introduction of the @snippet tag in Java 18, it is possible to include code snippets inside a javadoc. Code inside these snippets should use spaces only or tabs only (i.e., not a mix of both). Not doing so will lead to the inconsistent indentation in the javadoc output.

Noncompliant Code Example

{@snippet :
class HelloWorld {                               // Noncompliant - spaces and tabs are mixed
     public static void main(String... args) {
<TAB><TAB>System.out.println("Hello World!");
     }
}
}

Compliant Solution

Use spaces only

{@snippet :
class HelloWorld {
     public static void main(String... args) {
        System.out.println("Hello World!");
     }
}
}

Use tabulations only

{@snippet :
class HelloWorld {
<TAB>public static void main(String... args) {
<TAB><TAB>System.out.println("Hello World!");
<TAB>}
}
}
