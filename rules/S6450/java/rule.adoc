Either use spaces only or tabs only in javadoc code snippets. Not doing so will lead to the inconsistent indentation in the javadoc output.


== Noncompliant Code Example ==
[source,java]
----
{@snippet :
class HelloWorld {                               // Noncompliant - spaces and tabs are mixed
     public static void main(String... args) {
<TAB><TAB>System.out.println("Hello World!");
     }
}
}
----


== Compliant Solution ==

=== Use spaces only ===
[source,java]
----
{@snippet :
class HelloWorld {
     public static void main(String... args) {
        System.out.println("Hello World!");
     }
}
}
----


=== Use tabulations only ===
[source,java]
----
{@snippet :
class HelloWorld {
<TAB>public static void main(String... args) {
<TAB><TAB>System.out.println("Hello World!");
<TAB>}
}
}
----
