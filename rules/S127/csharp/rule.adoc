=== Noncompliant code example

[source,csharp]
----
class Foo 
{
    static void Main() 
    {
        for (int i = 1; i <= 5; i++) 
        {
            Console.WriteLine(i);
            if (condition) 
            {
               i = 20;
           }
        }
    }
}
----

=== Compliant solution

[source,csharp]
----
class Foo 
{
    static void Main() 
    {
        for (int i = 1; i <= 5; i++) 
        {
            Console.WriteLine(i);
        }
    }
}
----
