== Noncompliant Code Example

----
func doSomething() {
  ;                                                       // Noncompliant
}

func doSomethingElse() {
  fmt.Println("doSomethingElse");;     // Noncompliant - double useless ;
  ...
}
----

== Compliant Solution

----
func doSomething() {
}

func doSomethingElse() {
  fmt.Println("doSomethingElse")
  ...
}
----
