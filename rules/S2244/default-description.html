Description

Why is this an issue?

Using the = operator to copy the content of an internal table is more efficient than using LOOP + APPEND statements.

Noncompliant code example

REFRESH ITAB2.
LOOP AT ITAB1 INTO WA.
  APPEND WA TO ITAB2.
ENDLOOP.

Compliant solution

ITAB2[] = ITAB1[]
