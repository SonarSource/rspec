=== Noncompliant code example

[source,cpp,diff-id=1,diff-type=noncompliant]
----
// C++14 attribute
[[deprecated]]
void fun();

// GNU attribute
__attribute__((deprecated))
void fun();

// Microsoft attribute
__declspec(deprecated)
void fun();

void example() {
  fun(); // Noncompliant
}
----
