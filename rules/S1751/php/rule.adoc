=== Noncompliant code example

[source,php]
----
for ($i = 0; $i < 10; $i++) { // Noncompliant
  echo "i is $i";
  break;
}
...
for ($i = 0; $i < 10; $i++) { // Noncompliant
  if ($i == $x) {
    break;
  } else {
    echo "i is $i";
    return;
  }
}
----

=== Compliant solution

[source,php]
----
for ($i = 0; $i < 10; $i++) {
  echo "i is $i";
}
...
for ($i = 0; $i < 10; $i++) {
  if ($i == $x) {
    break;
  } else {
    echo "i is $i";
  }
}
----
