There is no need to use the final modifier inside a final class. Everything in it is final by default.

Noncompliant Code Example

final class MyClass {

  public final String getName() {  // Noncompliant
    return name;
  }
}

Compliant Solution

final class MyClass {

  public String getName() {  // Compliant
    return name;
  }
}
