=== Noncompliant code example

With the default regular expression ``++^[A-Z]$++``:

[source,swift]
----
public class MyClass<TYPE> {    // Noncompliant
  func method<TYPE>(t : TYPE) { // Noncompliant
  }
}
----

=== Compliant solution

[source,swift]
----
public class MyClass<T> { 
  func method<T>(t : T) {
  }
}
----
