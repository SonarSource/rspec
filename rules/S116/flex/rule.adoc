== Noncompliant Code Example

With the default regular expression ``++^[_a-z][a-zA-Z0-9]*$++``:

[source,flex]
----
class MyClass {
  public var my_field:int;
}
----

== Compliant Solution

----
public class MyClass {
   public var myField:int;
}
----
