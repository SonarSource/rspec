=== Exceptions

This rule does not apply to `if` chains without `else`, nor to `switch` without a `default` clause.

[source,php]
----
if($b == 0) {    //no issue, this could have been done on purpose to make the code more readable
  doSomething();
} elseif($b == 1) {
  doSomething();
}
----
