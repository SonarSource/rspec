=== Exceptions

This rule does not apply to ``++if++`` chains without ``++else++``-s, or to ``++switch++``-es without ``++default++`` clauses.


----
if(b == 0) {    //no issue, this could have been done on purpose to make the code more readable
  doSomething();
} else if(b == 1) {
  doSomething();
}
----
