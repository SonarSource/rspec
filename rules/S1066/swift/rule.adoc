== Why is this an issue?

include::../description.adoc[]

Code like

[source,swift]
----
if condition1 {
  if condition2 {
    doSomething()
  }
}

if let y = someOptional {
  if x > 0 {
    doSomething()
  } 
}
----

Will be more readable as

[source,swift]
----
if condition1 && condition2 {
  doSomething()
}

if let y = someOptional where x > 0 {
  doSomething()
}

if x > 0, let y = someOptional {
  doSomething()
}
----
