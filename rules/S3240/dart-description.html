Why is this an issue?

In the interests of keeping code clean, the simplest possible conditional syntax should be used. That means using the ??= (conditional assignment) operator for a self-assign-if-not-null operation.

Noncompliant code example

if (a == null) {  // Noncompliant
    a = Object();
}

Compliant solution

a ??= Object();
