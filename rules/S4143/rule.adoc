=== Noncompliant code example

[source,java,diff-id=1,diff-type=noncompliant]
----
letters.put("a", "Apple");
letters.put("a", "Boy");  // Noncompliant

towns[i] = "London";
towns[i] = "Chicago";  // Noncompliant
----

=== Compliant solution

[source,java,diff-id=1,diff-type=compliant]
----
letters.put("a", "Boy");
letters.put("b", "Boy");

towns[i] = "Chicago";
----
