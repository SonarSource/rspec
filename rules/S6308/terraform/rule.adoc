== Sensitive Code Example

For https://registry.terraform.io/providers/hashicorp/aws/latest/docs/resources/opensearch_domain[aws_opensearch_domain]:

[source,terraform]
----
resource "aws_opensearch_domain" "opensearch" {
  encrypt_at_rest {
    enabled = false  # Sensitive, disabled by default
  }
}
----

== Compliant Solution

For https://registry.terraform.io/providers/hashicorp/aws/latest/docs/resources/opensearch_domain[aws_opensearch_domain]:

[source,terraform]
----
resource "aws_opensearch_domain" "opensearch" {
  encrypt_at_rest {
    enabled = true
  }
}
----
