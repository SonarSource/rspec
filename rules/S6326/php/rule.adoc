== Noncompliant Code Example

----
"/Hello,   world!/"
----

== Compliant Solution

----
"/Hello, {3}world!/"
----

== Exception

If the extended modife is set, whitespaces are ignored. In this case no issue should be triggered,
because the whitespaces may be intended to improve readability.
