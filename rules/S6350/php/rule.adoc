== Sensitive Code Example

----
$input = $_GET['input'];
system('php ' . escapeshellarg($input)); // Sensitive
----

== Compliant Solution

Use an allow-list to restrict the arguments to trusted values:
----
$input = $_GET['input'];
if (in_array($input, $allowed)) {
  system('php ' . escapeshellarg($input));
}
----
