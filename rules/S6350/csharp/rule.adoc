== Sensitive Code Example

----
Process.Start("cmd.exe", input); // Sensitive
----

== Compliant Solution

Use an allow-list to restrict the arguments to trusted values:
----
if (allowed.Contains(input)) {
  Process.Start("cmd.exe", input);
}
----
