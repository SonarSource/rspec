Description

Code snippets are useful for adding code examples to javadoc comments. This will help developers to better understand the usage of documented method. However, large code snippets might be confusing and can decrease the readability. So the general rule is to keep snippets as small as possible.

Sometimes there are cases where this is not possible to have a small snippet. For these cases we recommend to extract the snippet into a separate file and make it either an external or a hybrid one.

Compliant Solution

/**
 * The code prints "Hello, World!"
 * {@snippet :
 *  public static void main(String... args) {
 *      System.out.println("Hello, World!");
 *  }
 * }
 */

Or

{@snippet file=ExternalSnippet.java }
