Description

Why is this an issue?

In JCL, some statements require to have a label or name defined, otherwise it will trigger a JCL error. The statements that require a name/label are: * CNTL * JOB * NOTIFY * OUTPUT * in-stream PROC

How to fix it

Code examples

Noncompliant code example

//MYJOB   JOB DXXX
//        PROC
//STEP1   EXEC PGM=MYPGM
//        PEND

Compliant solution

//MYJOB   JOB DXXX
//TRPOC   PROC
//STEP1   EXEC PGM=MYPGM
//        PEND
