=== Code examples

The following noncompliant code:

[source,cpp,diff-id=1,diff-type=noncompliant]
----
void doSomething()
{
    // ...
    std::cout << "My Message";                // Noncompliant
    // ...
}
----

Could be replaced by:

[source,cpp,diff-id=1,diff-type=compliant]
----
void doSomething()
{
    // ...
    Log().Get(logINFO) << "My Message";
    // ...
}
----
