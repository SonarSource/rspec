== Noncompliant Code Example

----
std::cout << "My Message";                // Noncompliant
std::cerr << "My Message";                // Noncompliant
printf("My Message");                     // Noncompliant
----

== Compliant Solution

----
Log().Get(logINFO) << "My Message";
----
