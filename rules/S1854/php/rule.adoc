== Noncompliant Code Example

----
$i = $a + $b; // Noncompliant; calculation result not used before value is overwritten
$i = compute();
----

== Compliant Solution

----
$i = $a + $b;
$i += compute();
----

== Exceptions

This rule ignores initializations to -1, 0, 1, ``++null++``, ``++true++``, ``++false++``, ``++""++``, ``++[]++`` and ``++array()++``.
