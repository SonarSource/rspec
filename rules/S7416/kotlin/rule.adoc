== How to fix it in Android

=== Code examples

==== Noncompliant code example

[source,kotlin,diff-id=1,diff-type=noncompliant]
----
android {
    buildTypes {
        release {
            isDebuggable = true // Sensitive
        }
    }
}
----

==== Compliant solution

[source,kotlin,diff-id=1,diff-type=compliant]
----
android {
    buildTypes {
        release {
            isDebuggable = false
        }
    }
}
----
