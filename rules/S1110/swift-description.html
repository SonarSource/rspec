Description

Why is this an issue?

Useless parentheses can sometimes be misleading and so should be removed.

Noncompliant code example

return ((x + 1))       // Noncompliant
var x = ((y / 2 + 1))  // Noncompliant
if ((x > 0)) { ... }   // Noncompliant

Compliant solution

return (x + 1)
return x + 1
var x = (y / 2 + 1)
var x = y / 2 + 1
if (x > 0) { ... }
if x > 0 { ... }
