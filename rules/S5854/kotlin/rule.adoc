== Noncompliant Code Example

----
val str = "e\u0300"
val regex = Regex("é|ë|è") // Noncompliant
println(str.replace(regex, "e")) // prints 'è'
----


== Compliant Solution

----
val str = "e\u0300"
val regex = Regex("é|ë|è", RegexOption.CANON_EQ) // Noncompliant
println(str.replace(regex, "e")) // prints 'è'
----
