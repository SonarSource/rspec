== Sensitive Code Example

[source,ruby]
----
class ApplicationController < ActionController::Base
  protect_from_forgery except: [:example_endpoint] # Sensitive, CSRF protection is disabled for the example_endpoint
end
----

[source,ruby]
----
class ApplicationController < ActionController::Base
  skip_before_action :verify_authenticity_token # Sensitive
end
----

[source,ruby]
----
class ApplicationController < ActionController::Base
  skip_forgery_protection # Sensitive
end
----

== Compliant Solution

[source,ruby]
----
class ApplicationController < ActionController::Base
  protect_from_forgery
end
----
