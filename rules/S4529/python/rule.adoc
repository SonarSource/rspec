== Sensitive Code Example

Django endpoint declaration

----
from django.urls import path, re_path

def declare_views(views):
    return [
        path('', views['index']),  # Sensitive
        re_path(r'^about/[0-9]*/$', views['about']),  # Sensitive
    ]
----
