==== Noncompliant code example

[source,apex,diff-id=1,diff-type=noncompliant]
----
if (booleanMethod() || false) { /* ... */ }

----

==== Compliant solution

[source,apex,diff-id=1,diff-type=compliant]
----
if (booleanMethod()) { /* ... */ }

doSomething(booleanMethod());
----
