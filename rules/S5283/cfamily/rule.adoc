Variable length arrays should have a well-defined, positive size.


== Noncompliant Code Example

----
void f1() {
  int n;
  int a[n]; // Noncompliant; n is undefined
}

void f2() {
  int n = 0;
  int a[n]; // Noncompliant; array of zero size
}
----
