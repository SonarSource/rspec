=== Noncompliant code example

[source,kotlin]
----
fun pickNumber() : Int {
  var i = 0
  var j = 0

  i = i++ // Noncompliant; i is still zero

  return j++ // Noncompliant; 0 returned
}
----


=== Compliant solution

[source,kotlin]
----
fun pickNumber() : Int {
  var i = 0
  var j = 0

  i++ 
  return ++j 
}
----
