== Noncompliant Code Example

----
do {
    try loadImage(imagePath)
} catch { // Noncompliant
    throw error
}
----

== Compliant Solution

----
do {
    try loadImage(imagePath)
} catch {
    handleImageError(error)
    throw error
}
----
or

----
try loadImage(imagePath)
----
