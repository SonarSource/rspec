== Noncompliant Code Example

[source,cs]
----
int GetBestNumber()
{
  return 12;  // Noncompliant
}
----

== Compliant Solution

[source,cs]
----
const int BestNumber = 12;
----
or

[source,cs]
----
static readonly int BestNumber = 12;
----
