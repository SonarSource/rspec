Description

Why is this an issue?

If a class method doesn’t access properties of that class (i.e. it doesn’t use this), it should be made static.

Noncompliant code example

class Foo {
  constructor() {
    this.i = 1;
  }

  bar() {
    window.focus();
  }
}

Compliant solution

class Foo {
  constructor() {
    this.i = 1;
  }

  static bar() {
    window.focus();
  }
}

Exceptions

The constructor is exempt of this rule.
