== Noncompliant Code Example

[source,python]
----
def f(b):
    a = True
    if a:  # Noncompliant
        doSomething()

    if a and b:  # Noncompliant; "a" is always "True"
        doSomething()
----

== Compliant Solution

[source,python]
----
def f(b):
    a = True
    if foo(a):
        doSomething()

    if b:
        doSomething()
----
