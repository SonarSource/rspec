== Sensitive Code Example

The following code sample does not use integrity checks:

[source,html]
----
<script
    src="https://cdn.example.com/v5.3.6/script.js"
></script> <!-- Sensitive -->
----

The following code sample uses integrity checks, but no version pinning. A
change on the `latest` version will break it:

[source,html]
----
<script
    src="https://cdn.example.com/latest/script.js"
    integrity="sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ8K/uxy9rx7HNQlGYl1kPzQho1wx4JwY8wC"
></script> <!-- Sensitive -->
----

== Compliant Solution

[source,html]
----
<script 
    src="https://cdn.example.com/v5.3.6/script.js"
    integrity="sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ8K/uxy9rx7HNQlGYl1kPzQho1wx4JwY8wC"
></script>
----
