Description

Why is this an issue?

Using several "--" or "++" unary operators in the same arithmetic expression can quickly make the expression unreadable.

Noncompliant code example

var j:int = foo++ - --bar;

Compliant solution

bar--;
var j:int = foo++ - bar;

Implementation Specification

(visible only on this page)

Message

Split this expression into multiple expressions so that each one contains no more than a single "++" or "--" unary operator
