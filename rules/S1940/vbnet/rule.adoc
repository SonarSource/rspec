== Noncompliant Code Example

[source,vbnet]
----
If Not (a = 2) Then  // Noncompliant
Dim b as Boolean = Not (i < 10)  // Noncompliant
----

== Compliant Solution

----
If a <> 2 Then
Dim b as Boolean = i >= 10
----
