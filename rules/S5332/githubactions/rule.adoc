== Sensitive Code Example

[source,yaml,diff-id=1,diff-type=noncompliant]
----
name: Download Data

on: push

jobs:
  download:
    runs-on: ubuntu-latest
    steps:
      - name: Download file
        run: curl http://www.example.com/  # Sensitive
----

== Compliant Solution

[source,yaml,diff-id=1,diff-type=compliant]
----
name: Download Data

on: push

jobs:
  download:
    runs-on: ubuntu-latest
    steps:
      - name: Download file
        run: curl https://www.example.com/
----
