=== Noncompliant code example

[source,dart]
----
RegExp(r'([');
RegExp(r'(\\w+-(\\d+)');
----


=== Compliant solution

[source,dart]
----
RegExp(r'\(\[');
RegExp(r'(\\w+)-(\\d+)');
----
