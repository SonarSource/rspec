Description

Why is this an issue?

The Cyclomatic Complexity of methods should not exceed a defined threshold. Complex code may perform poorly and can be difficult to test thoroughly.

Implementation Specification

(visible only on this page)

Message

The Cyclomatic Complexity of this method "XXXXX" is Y which is greater than T authorized.

Parameters

Threshold
10

The maximum authorized complexity in methods.

Highlighting

  • primary: method name

  • secondary: each piece of code in the method that increments the complexity score (if, return, ||, …​).

    • message: +1
