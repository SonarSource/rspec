== Noncompliant Code Example

----
val sr = SecureRandom()
sr.setSeed(123456L) // Noncompliant
val v = sr.nextInt()
----
----
val sr = SecureRandom("abcdefghijklmnop".toByteArray(charset("us-ascii"))) // Noncompliant
val v = sr.nextInt()
----


== Compliant Solution

----
val sr = SecureRandom()
val v = sr.nextInt()
----
