=== Noncompliant code example

[source,kotlin]
----
val sr = SecureRandom()
sr.setSeed(123456L) // Noncompliant
val v = sr.nextInt()
----
[source,kotlin]
----
val sr = SecureRandom("abcdefghijklmnop".toByteArray(charset("us-ascii"))) // Noncompliant
val v = sr.nextInt()
----


=== Compliant solution

[source,kotlin]
----
val sr = SecureRandom()
val v = sr.nextInt()
----
