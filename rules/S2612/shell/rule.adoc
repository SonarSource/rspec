To be secure, remove the unnecessary permissions. If required, use `--chown` to
set the target user and group.


== Sensitive Code Example

[source,bash,diff-id=1,diff-type=noncompliant]
----
#!/bin/bash

# Sensitive
chmod +x resource
# Sensitive
chmod u+s resource
# Sensitive
chmod 0777 resource
----

== Compliant Solution

[source,bash,diff-id=1,diff-type=compliant]
----
#!/bin/bash

chmod u+x resource
chmod +t resource
chmod 0754 resource
----
