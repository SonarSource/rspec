== Why is this an issue?

This rule allows banning certain PHP functions.


=== Noncompliant code example

Given a parameter of ``++foo++`` for this rule:

[source,php]
----
$arr=array("apple", "pear","banana");
echo foo($arr);  // Noncompliant
----
