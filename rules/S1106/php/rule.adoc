== Noncompliant Code Example

----
function myMethod() {  // Noncompliant
  if(something) {  // Noncompliant
    executeTask();
  } else {  //Noncompliant
    doSomethingElse();
  }
}
----

== Compliant Solution

----
function myMethod() 
{
  if(something)
  {
    executeTask();
  } else 
  {
    doSomethingElse();
  }
}
----
