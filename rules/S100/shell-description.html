Description

Why is this an issue?

Shared naming conventions allow teams to collaborate efficiently.

This rule raises an issue when a method or a property name is not snake_cased.

How to fix it

Rename the function so that its name conforms to the snake_case convention.

Code examples

Noncompliant code example

function doSomething() { # Noncompliant
}

Compliant solution

function do_something() {
}
