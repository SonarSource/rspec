=== Noncompliant code example

[source,php]
----
// This can never match because $ and ^ have been switched around
preg_match("/$[a-z]+^/", $str); // Noncompliant
----


=== Compliant solution

[source,php]
----
preg_match("/^[a-z]+$/", $str);
----
