== Noncompliant Code Example

[source,php]
----
"/[0-9]/" // Noncompliant - same as "/\d/"
/[^0-9]/" // Noncompliant - same as "/\D/"
"/[A-Za-z0-9_]/" // Noncompliant - same as "/\w/" 
"/[\w\W]/" // Noncompliant - same as "/./" 
"/a{0,}/" // Noncompliant - same as "/a*/" 
----

== Compliant Solution

[source,php]
----
"/\d/"
"/\D/"
"/\w/"
"/./"
"/a*/"
----
