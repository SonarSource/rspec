== Sensitive Code Example

----
$hash = md5($data); // Sensitive
$hash = sha1($data);   // Sensitive
----

== Compliant Solution

----
// for a password
$hash = password_hash($password, PASSWORD_BCRYPT); // Compliant

// other context
$hash = hash("sha512", $data);
----
