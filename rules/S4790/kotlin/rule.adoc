=== Code examples

==== Noncompliant code example

[source,kotlin,diff-id=1,diff-type=noncompliant]
----
val md5: MessageDigest = MessageDigest.getInstance("MD5");  // Noncompliant
// Note: SHA is not a standard name, for most security providers it's an alias of SHA-1
val sha: MessageDigest = MessageDigest.getInstance("SHA");  // Noncompliant
val sha1: MessageDigest = MessageDigest.getInstance("SHA1");  // Noncompliant
----

==== Compliant solution

[source,kotlin,diff-id=1,diff-type=compliant]
----
val sha512: MessageDigest = MessageDigest.getInstance("SHA-512");
----
